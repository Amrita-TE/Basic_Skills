# 🚀 Strict Engineering Discipline Skill for Cursor

## 📌 Overview
This repository contains a custom **Cursor skill (`Basic_Skill.md`)** designed to enforce disciplined, precise, and production-safe AI-assisted development.

The skill ensures that the agent follows a structured workflow focused on **clarity, minimal changes, and correctness**, reducing errors and unnecessary modifications.

---

## 🎯 Purpose
The goal of this skill is to guide AI agents to:
- Work with clear intent and direction
- Avoid assumptions and ambiguity
- Produce clean and minimal code changes
- Deliver reliable, predictable results

---

## 🧠 Core Principles

### 1. Plan Before Modifying
The agent must define a clear plan before making any changes.

### 2. No Assumptions
If anything is unclear, the agent must ask for clarification instead of guessing.

### 3. Simplicity First
Solutions should be simple, practical, and focused only on the current problem.

### 4. Surgical Changes
Only required lines or files should be modified—no unnecessary edits or refactoring.

### 5. Goal-Driven Execution
The agent must define the goal, execute changes, and verify completion before finishing.

---

## ⚙️ Workflow Enforced

1. Understand the task  
2. Ask clarifying questions (if needed)  
3. Create a clear plan  
4. Apply minimal changes  
5. Verify the goal is achieved  

---

## ✅ Benefits

- Prevents unnecessary code changes  
- Reduces risk in production environments  
- Improves consistency in AI-assisted development  
- Eliminates over-engineering  
- Ensures predictable and reliable outputs  

---

## 📂 Usage

1. Add the `skill.md` file to your Cursor configuration  
2. Enable the skill for your workspace  
3. Start using Cursor with stricter, more disciplined AI behavior  

---

## 💡 Ideal Use Cases

- Production support and bug fixes  
- Code review assistance  
- Controlled refactoring  
- Data engineering workflows (e.g., pipelines, ETL jobs)  

---

## 📖 Summary
This skill transforms Cursor into a **disciplined engineering assistant** that prioritizes simplicity, precision, and correctness—making it a reliable partner for real-world development workflows.
